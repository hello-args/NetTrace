# Synchronous IP Lookup Tool

A synchronous version of the IP lookup tool using `requests`, designed to fetch IP metadata from `ip-api.com`. Simple, robust, and extensible for quick use cases with fewer IPs.

---

## 🔧 Features

- ✅ Simple synchronous implementation
- 🔁 Retry mechanism for transient errors
- 📊 Excel read/write with `pandas` and `openpyxl`
- 🧱 Modular code with service, config, and handler separation
- 🪵 Logging of warnings and errors per module
